# Security Policy

This public repository is a documentation and citation surface for Grey Mirror. It does not contain the proprietary JustLayMe application source, production model code, customer uploads, private datasets, infrastructure secrets, or deployment configuration.

## What To Report

Please report security issues involving:

- Accidental exposure of private user data in this public repository.
- Broken links that appear to request credentials or private uploads.
- Files that accidentally disclose secrets, internal service URLs, tokens, keys, or production configuration.
- Misleading public security claims that should be corrected.

## What Not To Submit Here

Do not open a public issue containing:

- Real relationship-message exports.
- Authentication credentials.
- API keys, tokens, or secrets.
- Private user reports.
- Exploit details against the live JustLayMe service.

Use the official JustLayMe support/security contact listed at <https://justlay.me/> for sensitive reports.

## Supported Surface

This repository supports public documentation files only. The live product, user accounts, billing, uploads, reports, and private APIs are managed outside this repository.

## Disclosure Guidance

If you believe a live-service vulnerability exists, send a private report through the official JustLayMe contact path. Do not publish exploit steps or private data in GitHub Issues, pull requests, discussions, or comments.
