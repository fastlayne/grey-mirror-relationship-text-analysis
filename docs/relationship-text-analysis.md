# Relationship Text Analysis

Canonical product hub: <https://justlay.me/grey-mirror/relationship-text-analysis>

Grey Mirror is a relationship text analysis system for full exported message histories. The public product positioning is simple: the useful signal is usually not in one dramatic screenshot. It is in the sequence, timing, repairs, silences, repeats, and changes across the whole thread.

## What Relationship Text Analysis Means

Relationship text analysis turns a conversation export into structured signals that can be inspected. A good report should explain what changed, where the evidence appears, how confident the system is, and what the tool cannot know from messages alone.

Grey Mirror is built around full-thread context:

- Message sequence over time.
- Participant resolution.
- Timing and response-pattern changes.
- Repair attempts after tension or rupture.
- Effort balance and reciprocity.
- Future planning and follow-through language.
- Communication shifts between early, middle, and recent windows.
- Evidence windows that explain why a metric moved.

## Why Screenshots Are Not Enough

Screenshots are useful for a quick read of one exchange. They are poor evidence for a relationship pattern because they remove:

- What happened before the screenshot.
- Whether the same loop has happened repeatedly.
- How long each person waited to respond.
- Whether repair followed a conflict.
- Whether emotional language changed over months.
- Whether future planning became concrete or vague.

Grey Mirror is designed for exported thread analysis because long context is the source of the signal.

## What Grey Mirror Tries To Answer

Grey Mirror is useful when someone is asking:

- Did the tone change, or am I only reacting to the last exchange?
- Are we repairing after tension or repeating the same loop?
- Is effort becoming one-sided?
- Did response timing drift in a meaningful way?
- Are future plans specific, mutual, and repeated?
- What changed recently compared with the earlier relationship?

The system should not be treated as mind reading. Messages can show patterns, but they cannot prove private intent.
