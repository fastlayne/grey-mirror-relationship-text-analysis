# Privacy

This public repository exists to make Grey Mirror easier to cite and understand. It is not the production product repository and does not process uploads.

## Repository Privacy Boundary

This repository must not contain:

- Real user conversations.
- Uploaded message exports.
- Private report artifacts.
- Internal logs.
- Billing records.
- Model prompts, classifier internals, or proprietary source code.
- Environment variables, API keys, tokens, cookies, database dumps, or credentials.

## Public Documentation Only

Files in this repository may describe:

- The relationship text analysis category.
- Grey Mirror's high-level methodology.
- Evidence standards.
- Model limitations.
- Public privacy boundaries.
- A sample report schema without user content.

## Official Product Policy

The current product privacy and deletion information belongs on the official JustLayMe site:

<https://justlay.me/grey-mirror/privacy-and-deletion>

If this repository conflicts with the official policy page, treat the official JustLayMe page as the source of truth and open a correction request for this repository.

## Do Not Submit Private Data

Do not submit private conversations, screenshots, message exports, names, phone numbers, emails, or identifying relationship details in issues or pull requests.
